# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this is

A Teardown mod adding a gasoline flamethrower tool: adjustable nozzle velocity (via a physical knob, arrow keys by default), an ammo system, and a custom fire simulation that spawns fires along the flame trajectory with `SpawnFire()` rather than relying on the engine's built-in flame behavior. Published on Steam Workshop (ID in `mod/id.txt`).

## Development

Pure Lua, no build/test/lint toolchain. The mod content lives in `mod/` — to iterate, copy or symlink `mod/` into Teardown's mods folder and reload in-game.

- `teardown.d.lua` provides Teardown API type stubs for IDE autocomplete (generated by the sibling `teardown-api-stub-generator` repo).
- Debug mode: uncomment `Debug:init()` in `mod/main.lua` (toggled in-game with HOME). Adds raycast visualization and property watches. Keep it disabled in releases.

## Architecture

`mod/main.lua` is the entry point. Scripts are composed with Teardown's `#include` preprocessor directives (not `require()`). Lifecycle: `init()` registers the tool, dependencies initialize lazily on first `tick()`, and `update()` (tool active + usable) drives flame spawning and ammo consumption.

OOP via Lua tables with metatables. Flow: holding `usetool` → `Flamethrower:update()` (`mod/scripts/flamethrower.lua`) → `Nozzle:throwFlames()` (`mod/scripts/nozzle.lua`) raycasts and creates `Flame` entities (`mod/scripts/entities/flame.lua`), which each frame advance and spawn fires along their path. `Knob` (`mod/scripts/knob.lua`) adjusts flame velocity (5–25 m/s). `SoundManager` handles start/loop/end sounds. `mod/options.lua` is a separate script Teardown loads for the settings UI (keybind remapping, fire-limit override, inventory slot).

Settings persist via Teardown registry keys under `savegame.mod.features.*`; defaults are only written when the key is unset.

## Gotchas

- Voxel shapes are accessed by hardcoded index (`shapes[2]` = knob, `shapes[3]` = fire starter, `shapes[4]` = nozzle). Editing `mod/vox/Flamethrower.vox` can silently break these bindings.
- The fire-limit option mutates global game state (`game.fire.maxcount`).
- Voxel size constant (0.05) lives in `mod/scripts/lib/engine.lua` and is used throughout transform math.
